FitNurture: Advanced Posture Detection

Harnessing Computer Vision & AI to Analyze and Understand Posture

What is FitNurture?

FitNurture is an innovative application designed to analyze human posture using cutting-edge computer vision technology. By processing images or live camera feeds, it meticulously identifies key body landmarks to detect potential postural abnormalities.

The primary goal of FitNurture is to provide users—particularly for assessing children's posture—with valuable insights into common postural issues. It offers an intuitive, user-friendly interface to capture visual data, receive a comprehensive analysis, and obtain general recommendations. Now, supercharged with **✨ AI-Powered Personalization**, FitNurture also delivers tailored exercise and lifestyle suggestions using Google's advanced Gemini Large Language Model.

How FitNurture Works: A Step-by-Step Guide

1. Image Input

Users can easily upload existing images (JPG, PNG, JPEG) or utilize a connected camera to capture new photos directly within the application.

2. CV Analysis

FitNurture employs MediaPipe Pose estimation to accurately detect crucial body landmarks. It then calculates various biomechanical metrics from these landmarks.

3. ✨ AI Suggestions

The integrated Gemini API generates personalized, actionable exercise and lifestyle tips based on the specific postural findings.

4. Results & Reporting

The app presents detected abnormalities, standard recommendations, AI-powered tips, displays landmarked images, and allows for PDF report generation and data storage.

Key Features of FitNurture

Flexible Input Options

  • Image Upload: Seamlessly supports standard image formats (JPG, PNG, JPEG).
  • Camera Capture: Allows direct photo capture using the device camera for on-the-spot analysis.

Comprehensive Analysis Modes

  • Single View Analysis: Optimized for side-view images to detect common sagittal plane postural issues.
  • Multi-View Analysis: Utilizes four distinct views (Front, Left Side, Right Side, Back) for a more holistic and robust postural assessment.

✨ AI-Powered Personalized Suggestions

Unlocking a new level of guidance, FitNurture leverages the power of Google's Gemini API. This advanced Large Language Model generates tailored exercise routines, specific stretches, and actionable lifestyle advice. These suggestions are crafted based on the unique postural abnormalities detected for each individual, providing a more personalized and effective path to posture improvement.

Detailed Reporting

  • PDF Reports: Generates comprehensive PDF reports including child details, processed landmarked images, detected issues, standard recommendations, and the new AI-powered personalized tips.
  • CSV Export: Allows users to download all locally saved records, including AI suggestions if saved, in CSV format for external analysis or record-keeping.

Data Management

  • Local Storage: Option to save complete analysis results, including any generated AI tips, locally within the app for easy access.
  • Cloud Upload: Securely uploads core postural records to a SQL database for persistent storage and wider accessibility (configurable to include AI tips if DB schema allows).

Posture Issues FitNurture Can Identify

FitNurture is meticulously trained to identify a range of common postural abnormalities. Users have the flexibility to select which specific issues they want the application to screen for during the analysis:

Kyphosis Lordosis Tech Neck Scoliosis Flat Feet Gait Abnormalities Knock Knees Bow Legs

For each detected issue, FitNurture provides an indication of its presence based on calculated biomechanical metrics and predefined thresholds. It offers general standard recommendations and now, with Gemini integration, highly personalized AI-powered suggestions for corrective actions.
